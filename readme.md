   How to run the project

1. In terminal run the command:  
   `git clone https://github.com/rolling-scopes-school/gryzun33-JS2020Q3.git`

2. Go to folder "gem-puzzle"

3. In terminal run command:  
   `npm install`

4. In terminal run command: 
   `npm run dev`

5. Follow the link:  
  `http://localhost:9000/`
